About NullOS

NullOS is a pure command-line development environment for Linux, designed for speed, precision, and absolute control. It features its own shell (null), an IP scanner (ip_check), and a growing suite of native utilities.

Features

null Shell — Custom Bash-like CLI with built-in commands
ip_check — Multi-threaded LAN IP scanner
Null-project — C/C++ project manager
ELF tool integration
100% bloat-free, written in C and Shell
